Этот плагин добавляет каталог видеороликов из UROK.su на ваш сайт.

== Description ==

С помощью этого плагина вы можете добавить каталог видеороликов к себе в блог.

Соответственно автоматически формируются рубрики Вордпресса (аналогичные видеоканалам http://UROK.su)
Автоматически добавляются теги, что повышает привлекательность вашего блога у поисковых систем.

В папке EXAMPLE также помещен демонстрационный файл сайдбара для организации ТОП 10 Самых популярных
видеороликов.

Пример ТОП 10 Самых популярных видеороликов вы можете посмотреть на демо-сайте http://demo.urok.su

== Installation ==

Процедура установки:

Процедура установки проста:

1) После разархивации копируем папку add-uroksu-catalog в папку plugins Вордпресса
2) Активируем его в разделе "Плагины" аминистративной панели Вордпресс.
3) Затем переходим в раздел "Параметры" и нажимаем UROK.su
4) Теперь нажимаете "Добавить видеоролики".

И все! Открывайте ваш вордпресс и любуйтесь.

== Frequently Asked Questions ==

Вопрос: Как часто запускать плагин?
Ответ:  Не рекомендуем чаще одного раза в сутки.
        При этом добавляются новые видеоролики и обновляются рейтинги существующих.
